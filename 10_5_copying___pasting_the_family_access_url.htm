10.5 Copying & Pasting the Family Access URL

When you are ready to solicit screenings and invite caregivers to enter screening results online, you can copy the Family Access URL and paste it to your webpage, an email, etc.; however you are contacting caregivers.

To copy and paste the Family Access URL:

  1. Select Family Access from the top navigation menu.
  2. Highlight the string "http://www.asqonline.com/family/xxxxxx" where the x's contain a code that is specific to your organization and questionnaire.
  3. Copy this URL by right clicking and selecting "Copy".(Or, Ctrl + C for PCs; Command + C for Macs.)
  4. Paste this URL where desired, by right clicking and selecting “Paste.” (Or, Ctrl + V for PCs; Command + V for Macs.)
