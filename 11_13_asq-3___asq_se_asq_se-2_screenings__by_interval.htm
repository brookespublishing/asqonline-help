11.13 ASQ-3 Screenings, by Interval

The ASQ-3 Screenings, By Interval report shows the total number of screenings in your program for each age interval.

These reports can be narrowed down by interval(s), status, language, Provider, and screening date range.

 

To run the ASQ-3 Screenings, by Interval Report:

 

  1. Select Reports from the top navigation menu.
  2. Select either ASQ-3 Screenings, by Interval.
  3. Enter your report generation criteria (leave a field blank to include all):
    1. Interval(s)
    2. Status (Active or Archived)
    3. Language (English or Spanish)
    4. Provider
    5. Screening date range
  4. Click Generate Report. The page will display a “report status” while the report is being processed.
  5. Three report generation options will appear: Download PDF, Download CSV, or Preview Report. Select the report generation option that you would like to run. (See Section 11.1 Report Generation Options for more information on each format.)
