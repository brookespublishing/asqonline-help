11.9 ASQ-3 Screenings Results, by Program

The ASQ-3 Screening Results, by Program Report lists the results of each individual child’s screenings within your program.  This report also shows any screening notes.

The ASQ-3 Screening Results, by Program Report lists the name of the child, the screening interval, screening date and results (Above, Monitoring, Below Cutoff) for each area of the ASQ-3 screening. For the Overall area, it shows you the number of items that the caregiver marked “Concern” and the number of items that were marked “No Concern.”

The reports can be narrowed down by interval, result, concern, status, language, Provider, and screening date range.

To run the ASQ-3 Screening Results, by Program Report:

 

  1. Select Reports from the top navigation menu.
  2. Select either ASQ-3 Screening Results, by Program.
  3. Enter your report generation criteria (leave a field blank to include all):
    1. Interval(s)
    2. Result

                                                              i.      Above Cutoff

                                                            ii.      Monitoring

                                                          iii.      Below Cutoff

    1. Concern (Concern or No Concern)
    2. Status (Active or Archived)
    3. Language (English or Spanish)
    4. Provider
    5. Include all child profile fields
    6. Include child profile custom fields 
    7. Show only latest screening
    8. Child
    9. Screening date range
  1. Click Generate Report. The page will display a “report status” while the report is being processed.
  2. Three report generation options will appear: Download PDF, Download CSV, or Preview Report unless one of the criteria selected is Child profile or Child custom field. Select the report generation option that you would like to run. For reports including Child profile or Child custom field the PDF option is not available. (See Section 11.1 Report Generation Options for more information on each format.)
