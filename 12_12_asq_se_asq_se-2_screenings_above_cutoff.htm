12.12 ASQ-3 Screenings Below Cutoff, by Area

The ASQ-3 Screenings Below Cutoff, by Area report shows the total number of screens that fell below cutoff (Concern) for your program.

This report lists how many screenings have been finalized, and how many of those screenings showed concern (Below Cutoff) in All Areas, One Area, Two Areas, or Three or More Areas. It further breaks it down by listing all areas (Communication, Gross Motor, Fine Motor, Problem Solving, and Personal-Social) and how many screenings fell below cutoff for each area. In addition, it shows you the total number of Overall Items that were marked “Concern” or “No Concern.”

This report can be narrowed down by status, language, Providers, and screening date range.

 

To run the ASQ-3 Screenings Below Cutoff, by Area Report:

 

  1. Select Reports from the top navigation menu.
  2. Select ASQ-3 Screenings Below Cutoff, by Area.
  3. Enter your report generation criteria (leave a field blank to include all):
    1. Status (Active or Archived)
    2. Language (English or Spanish)
    3. Provider(s)
    4. Screening date range
  4. Click Generate Report. The page will display a “report status” while the report is being processed.
  5. Three report generation options will appear: Download PDF, Download CSV, or Preview Report. Select the report generation option that you would like to run. (See Section 11.1 Report Generation Options for more information on each format.)
