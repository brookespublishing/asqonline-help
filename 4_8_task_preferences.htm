4.8 Task Preferences

The task preferences feature provides Program Administrators the ability to set automatic screening tasks for Child Profiles, so that follow ups are done per the standards of the program. Program Administrators can set preferences so that tasks are automatically created for Child Profiles when:
  • The child reaches a certain age
-OR-
  • The child needs follow up based on their screening result

Below are the various options for screening preferences that can be created to meet your program’s specific screening needs. The options are as follows:  
  • Automatic tasks based on the child’s age
  Example: “Create Referral task due when a child is [x] months and [x] days old.”  
  • Automatic tasks based on the child’s screening results
  Example: “Create Team Meeting, for ASQ-3 screenings that are below cutoff due  [x] days
