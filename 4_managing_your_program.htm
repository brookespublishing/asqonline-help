4 Managing Your Program

The Program page of ASQ Online is where you will go to set up your program and manage all aspects of your program.

It is within the Program page of ASQ Online that you can view and edit your program details, add all program users, enter your ASQ product keycodes to gain access to your questionnaires electronically, and view and add program notes and files, as well as custom tasks. The Program page includes the following sections:

  • Program Details 
  • Recent Program Users
  • Keycodes
  • Program Notes  
  • Program Files
  • Custom Tasks
