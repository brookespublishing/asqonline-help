5.4 Editing a Program's Profile Details

To edit a program’s profile details:
  1. Select Programs from the top navigation menu.   
  2. Search for a program by entering the program name, and/or the Administrator’s last name/first name, and click the Search button.  
 OR
  1. Scroll through the list of programs under the Programs tab. 
  2. Select the name of the program whose details you want to edit. 
  3. Click the Edit tab to the right of the Program Details tab. 
  4. Make all desired changes and click Save.
