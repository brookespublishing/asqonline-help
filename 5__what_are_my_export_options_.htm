5. What are my export options?

You can choose one of the following options when exporting data:

  • All: Selecting this option will export all data, including the Child Profile with associated Caregiver Profiles, Child Profile notes, Caregiver Profile notes, and all associated screenings for the child records selected. Any custom fields for the Child and Caregiver Profiles are also included in the export.
  • Child Profile Data Only: This option will export the Child Profile only (demographic information and any associated custom fields) for the child records selected.
  • Caregiver Data Only: Selecting this option will export the Caregiver Profile only (demographic information and any associated custom fields) for the child records selected.
  • Screening Data Only: This option will export all ASQ-3/ASQ:SE screenings, including the raw scores, cutoff scores, and responses to individual items for the child records selected.
  • You can also export the results of a search of the list of child profiles, the list of caregiver profiles and the list of screenings (this export option is for Program Administrators only).
