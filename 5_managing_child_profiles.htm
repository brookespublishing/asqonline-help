5 Managing Child Profiles

The Child Profiles page contains a list of all Child Profiles that you have added to the system, or that have been assigned to you. Each child has their own individual Child Profile page. A child’s profile page includes: the child’s profile details; the child’s screenings; the child’s tasks; notes pertaining to the child; the child’s caregivers; a list of the child’s Providers and Reviewers; and, if applicable, the child’s transfer history.

As a Provider, you can create Child and Caregiver Profiles, enter screening results, and add tasks and notes for the child.
