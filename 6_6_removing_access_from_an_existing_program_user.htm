6.6 Removing Access from an Existing Program User

If a user within your program has access to two or more user roles, it is possible for you to remove that user’s access to one of the roles, if needed.  

To remove user access:
  1. Select Programs from the top navigation menu. 
  2. Locate and select the name of the program the user is associated with. You will be directed to the Program Details page. 
  3. Click View All next to the Recent Program Users tab, or select the Search Program Users Quick Link to see a list of all users in the program. 
  4. Click on the user’s name to access his or her profile.   
  5. On the User Details page, there will be a list of all roles this user has access to under My Access List at the bottom of the page. 
  6. Select the role you would like to remove, and click on Remove Role, beneath the "Actions" column.  
  7. A pop-up message will appear, asking if you are sure. Click OK.


You can also remove multiple user access at once by following steps 1-4 and then completing the following steps:
    1. Click Edit.
    2. Scroll down to the Role section.
    3. Deselect all of the boxes for the roles the user should no longer have access to.
    4. Click Save.
