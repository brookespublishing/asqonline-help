7.2 Custom Fields

In the Pro account the Program Administrator has the ability to create custom fields that appear on either the Child Profiles or Caregiver Profiles of ASQ Online. Once these fields are created, they will automatically appear in the Child and Caregiver Profile Data Template and the Caregiver Profile Data Only Template.
