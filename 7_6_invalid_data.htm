7.6 Invalid Data

When transferring a child, if the child’s profile has invalid data, the page will state “Invalid data prevents this transfer from being completed. Please resolve the errors on the following records.” It will list the invalid data and the reason the data is invalid, which must be corrected before the Child Profile can be transferred. Contact your Program Administrator to correct the issues.

An example of an invalid data message you might see is:

“The caregiver profile 'Doe, John' is invalid: Email is invalid"
