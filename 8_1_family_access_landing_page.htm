8.1 Family Access Landing Page

The Family Access page in ASQ Online is where you are able to create your Family Access landing page. The landing page is a page that a user will visit (or “land on”) when they access the URL provided to them by the program. It is through the landing page that caregivers enter screening information for children.

The Family Access landing page includes the following information: program logo, program address information, welcome message, ASQ-3, ASQ:SE, ASQ:SE-2 questionnaire information summary, optional passcode field, consent paragraph, and optional footer logos.
