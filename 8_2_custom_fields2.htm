8.2 Custom Fields(Enterprise account only)

The Account Administrator has the ability to create custom fields that appear on either the Child Profiles or Caregiver Profiles of ASQ Online. Once these fields are created, they will automatically appear in the Child and Caregiver Profile Data Template and the Caregiver Profile Data Only Template.  For more information on custom fields see section 4.5.
