8 Import

As Account Administrator, you have the ability to import child, caregiver, and screening data directly into your ASQ Online account, as well as export child, caregiver, and screening data.

You can import data using Data Templates available directly from ASQ Online. The Data Templates can also be used to update existing child and caregiver data.
